# 🎤 Interview Assist AI

Interview Assist AI is an intelligent web-based application designed to help students prepare for technical interviews through AI-powered question generation and answer practice.

It provides a structured interview preparation environment with real-time interaction and performance tracking.

---

## 🚀 Current Features

* AI-based technical interview question generation
* Real-time answer practice interface
* User Authentication (Login & Register)
* Interview history tracking
* Clean and responsive dashboard UI
* MVC based backend architecture
* Secure API integration using environment variables
